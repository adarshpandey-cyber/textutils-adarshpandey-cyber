TextUtils [UPDATED]

Requirements:
1. You must have python installed.(Install it by going to python.org and downloading its latest version)
2. You must have Django installed.(Once you have python installed, you can easily install it by typing this command: pip install django)
3. A Python IDE(Optional).

How to use this website:-
1. Open your terminal in the folder where "manage.py" file is located.
2. Type this command in the terminal: python manage.py runserver
3. Your server is live now and access it via your browser at the address: localhost:8000
4. You can close your server by typing Ctrl+C in your terminal
